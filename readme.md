## Finance App

Stack:
- FastAPI
- SQLAlchemy
- SQLite
- React (Vite)
- Recharts

Features implementadas:

- CRUD de transações
- Categorias com tipo (income / expense)
- Dashboard com resumo financeiro
- Gráfico de gastos por categoria
- Filtro de categorias baseado no tipo de transação
- Separação de receitas e despesas

Próximos passos:

- contas financeiras
- orçamento por categoria
- gráficos mensais
- autenticação de usuários
